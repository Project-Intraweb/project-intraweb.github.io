Installation Guide

Before you start:

  • Visit Kiwix and download a zim. Put it into home folder as wiki.zim
  • You must have ethernet connected to the Pi, or wlan1. The access point will be wlan0.
  • You will need to enable swap if your Pi has 1GB of RAM. This was tested on a Raspberry Pi 4 with 4GBs of RAM.
  • Recommended: An external hard drive for Plex and Nextcloud.

Step 1

Ensure git is installed by running
sudo apt -y install git

Step 2

Clone the repository by running
git clone https://github.com/Project-Intraweb/Intraweb.git

Step 3

Run the installation script. Run cd Intraweb && sudo bash installer.sh
